    RandomSeed

    new RandomSeed(lower, upper, initCache, loop)

    带缓存的随机数种子。

    Source:

    Parameters:
    Name Type Default Description
    lower Number

    upper Number

    上限（包含）。
    initCache Array

    初始缓存数组。
    loop Boolean true
